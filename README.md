# VideoLengthCalculator
<br/>

Basic Java application to calculate total duration of all video files inside a directory and it's subdirectories.  <br/> <br/> 


**Dependencies** :  
<br/>
  i  . Java JDK build 1.8 <br/>
  ii . FFmpeg library  <br/>
  <br/>
